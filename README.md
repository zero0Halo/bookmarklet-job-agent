# bookmarklet-job-agent

A bookmarklet that works in conjunction with job-agent. Actual logic is in the `bookmarklet.js` file.

## To Create the Bookmarklet

Creates the `output.js` and `demo.html` files in the `dist` directory. Open the `demo.html`
file and follow its instructions.

```bash
npm run build
```

## To Create Only `output.js`

Creates only the `output.js` file in the `dist` directory.

```bash
npm run build:bookmarklet
```

## To Create Only `demo.html`

Creates only the `demo.html` file in the `dist` directory.

```bash
npm run build:demo
```
